# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good faith. Harassment,
personal attacks, and discriminatory behavior are not tolerated anywhere in
this project's spaces (issues, PRs, discussions).

Reports: email arorashreshth1@gmail.com. All reports are handled
confidentially.
